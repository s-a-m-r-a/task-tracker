<h2>📝 To-Do List</h2> <br>
A simple and interactive To-Do List web app built with HTML, CSS, and JavaScript. 🚀
<hr>
<h2>✨ Features</h2><br>
➕ Add new tasks <br>
✏️ Edit and remove tasks<br>
🔀 Sort tasks in ascending or descending order<br>
🧹 Clear input fields with a button<br>
